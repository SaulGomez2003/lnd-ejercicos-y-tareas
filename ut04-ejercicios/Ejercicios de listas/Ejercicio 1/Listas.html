HTTP
Protocolo de comunicación que permite la transferencia de información en la web. Facilita la conexión entre clientes y servidores.
HTML
Lenguaje de marcado utilizado para la creación de páginas web. Define la estructura del contenido mediante etiquetas.
URL
Dirección única que identifica un recurso en Internet, como una página web, imagen o documento.
TCP/IP
Conjunto de protocolos que permite la comunicación entre computadoras en una red. TCP gestiona la transmisión de datos y IP se encarga de la dirección y entrega de paquetes.
Internet
Red global de computadoras interconectadas que permite la comunicación y el intercambio de información a nivel mundial.
W3C
Organización internacional que desarrolla estándares y directrices para garantizar el crecimiento y accesibilidad de la web.
Mozilla Foundation
Organización sin fines de lucro dedicada al desarrollo de software libre y abierto, incluyendo el navegador Firefox, con el objetivo de mejorar la accesibilidad y seguridad en la web.
